Resume Screening & Candidate Ranking System
This project uses Machine Learning and NLP techniques to screen resumes and rank candidates based on job role fit.
Features:
* Resume text processing
* Job description matching
* Candidate ranking using similarity scores
* Resume-to-role comparison
* Match score visualization
Tools Used:
* Python
* Jupyter Notebook
* Scikit-learn
* TF-IDF Vectorization
* Cosine Similarity
This system helps recruiters shortlist candidates faster by matching resume skills with job requirements.
